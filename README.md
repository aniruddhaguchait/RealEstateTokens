# 🏠 Real Estate Tokenization on Stellar (Soroban)

## 📌 Project Description

Real Estate Tokenization is a blockchain-based solution that enables fractional ownership of real estate properties using the Stellar Soroban smart contract platform. Instead of requiring large capital to invest in property, users can purchase small tokenized shares representing ownership.

This project demonstrates how real-world assets (RWA) like property can be digitized and made accessible, transparent, and tradable on-chain.

---

## ⚙️ What It Does

* Tokenizes a real estate property into multiple digital shares (tokens)
* Allows users to buy fractional ownership of a property
* Stores ownership records securely on-chain
* Ensures transparent and tamper-proof property data

### 🔄 Flow:

1. Property owner initializes the property with total tokens
2. Each token represents a fraction of the property
3. Users purchase tokens via the smart contract
4. Ownership is recorded on-chain
